Setup in 30 Seconds

1

Download the Steering File

Get appian-locust-steering.md from the repository

curl -O https://raw.githubusercontent.com/mukundanramesh/appian-locust-support/main/appian-locust-steering.md
2

Copy to Your Project

Place it in your project's .kiro/steering directory

mkdir -p .kiro/steering mv appian-locust-steering.md .kiro/steering/

Or copy to ~/.kiro/steering/ to use across all projects

3

Start Using!

Open Kiro and ask about Appian Locust

See It In Action

You: How do I write an Appian Locust test?
Kiro: Here's a basic Appian Locust test structure:
from locust import HttpUser, task from appian_locust import AppianTaskSet class MyTaskSet(AppianTaskSet): @task def my_test(self): # Navigate to a site self.appian.visitor.visit_site("My Site") # Interact with UI form = self.appian.visitor.visit_record_type("MyRecord") form.click_button(label="Submit") class UserActor(HttpUser): tasks = [MyTaskSet] host = 'https://mysite.appiancloud.com' auth = ["username", "password"]

This creates a basic test that navigates to a site and interacts with a record. Would you like to see more advanced examples?

You: How do I debug if my test fails?
Kiro: Here are the key debugging techniques:
  1. Enable debug logging: Set log level to DEBUG
  2. Use test labels: More stable than display labels
  3. Check network tab: Inspect API calls in browser
  4. Verify selectors: Ensure UI elements exist

For more details, check the debugging guide at ReadTheDocs
